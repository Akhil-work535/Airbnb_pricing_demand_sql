# Airbnb Pricing & Demand Analytics (SQL)

## Business Problem
Airbnb wants to understand what factors influence booking demand in the Albany, NY market in order to optimize pricing strategies and improve host performance.

## Objective
Analyze listing-level data to identify:
- Pricing vs demand patterns
- Overpriced neighborhoods
- Host performance differences
- Impact of instant booking
- Effect of minimum night requirements

## Dataset
Source: InsideAirbnb.org  
Market: Albany, New York  

Files used:
- listings.csv
- calendar.csv

## Key Metric
Demand Score:
reviews_per_month / (365 - availability_365)

## Key Insights
- Demand drops significantly beyond $120 price range.
- Superhosts generate higher average demand.
- Entire homes outperform private rooms.
- Higher minimum night requirements reduce bookings.
- Multi-listing hosts show mixed performance.

## Tools
SQL (MySQL)

- Data import script
- Data cleaning script
- End-to-end analysis queries
